    • Marbles On Stream

    A live service multiplayer idle racing game that connects streamers directly to their viewers developed in the Unreal Engine. My contributions included bringing functionality to the "tilted" game mode that adapted the traditional game in to an action puzzle game. I also worked on implementing StreamSix's backend in to the game, allowing for much larger lobbies while still maintaining fluid gameplay and connections. Lastly, I headed an R&D project for a hub world where players could manage their own factory to get in game currency they would eventually be able to exchange for cosmetics. I assured all features would work on both PC and Mobile.

    • The Bingo Room

    A stream interactive game designed to allow streamers to play bingo with their viewers developed in Unity. My contributions included a keybinding system using Unity's modern input system. This allowed users to not only use their keyboard but also peripherals such as stream decks to interact with the game. I also worked on the mobile application ensuring functionality and adding a vibrating feedback system.
